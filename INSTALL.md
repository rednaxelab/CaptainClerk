# ProConnect Grid Auto-Fill Installation Guide

Follow these steps to install the extension in Google Chrome:

## 1. Enable Developer Mode
1. Open Google Chrome.
2. Navigate to `chrome://extensions/` by typing it in the address bar or via **Settings > Extensions**.
3. In the top right corner, toggle the **Developer mode** switch to **ON**.

## 2. Load the Extension
1. Click the **Load unpacked** button that appears in the top left.
2. In the file picker, navigate to and select the root folder of this project (`ChromeAuto`).
3. Click **Select** or **Open**.

## 3. Verify Installation
1. You should see "ProConnect Grid Auto-Fill" in your list of extensions.
2. Ensure the toggle switch for the extension is enabled.

## 4. Usage
1. Open a tax return in **Intuit ProConnect Tax Online**.
2. Navigate to a "Quick Entry" grid.
3. Click into the first cell where you want to start entering data.
4. Copy your data from Excel or Google Sheets (TSV format).
5. Press `Cmd+V` (Mac) or `Ctrl+V` (Windows) to paste. The extension will automatically distribute the data across the grid cells.
